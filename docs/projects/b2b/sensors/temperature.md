# Plotting data using Temperature sensor

**The sensor used: BMP180 /  BMP280**

Using the visual code, the KuttyPy+ takes the sensor data from an I2C temperature sensor and plots it on a time graph. BMP180/280 is a pressure and temperature sensor.

**Activity**: The participants are asked to find the ambient temperature of the lecture hall.
